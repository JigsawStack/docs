### Body Parameters

<ParamField body="name" type="string" required>
  The name of the template.
</ParamField>

<ParamField body="content" type="string" required>
  The html or text content.
</ParamField>

### Examples

- Simple data

```javascript

// Sample data

{
  "name": {
    "first": "Michael",
    "last": "Jackson"
  },
  "age": 50
}

// Usage

`{{name.first}} {{name.last}} died at age {{age}}.`

// Output

`Michael Jackson died at age 50.`
```

- Data with array of strings.

> When looping over an array of strings, a `.` can be used to refer to the current item in the list.

```javascript

// Data

{
  "musketeers": ["Athos", "Aramis", "Porthos", "D'Artagnan"]
}


// Usage

{{#musketeers}}
 {{.}}
{{/musketeers}}

// Output

* Athos
* Aramis
* Porthos
* D'Artagnan

```

### Response

<ResponseField name="success" type="boolean">
  Indicates whether the call was successful.
</ResponseField>

<RequestExample>

```javascript Node.js
const endpoint = "https://api.jigsawstack.com/v1/email/template";
const options = {
  method: "POST",
  headers: {
    "Content-Type": "application/json",
    "x-api-key": "<your-api-key>", // Replace with your actual API key.
  },
  body: JSON.stringify({
    name: "Onboarding Email",
    content: "<p>hello </p>",
  }),
};
const result = await fetch(endpoint, options);
const data = await result.json();
```

```bash cURL
    curl --location --request POST 'https://api.jigsawstack.com/v1/email/template' \
    --header 'Content-Type: application/json' \
    --header "x-api-key: <your-api-here>" \
    --data '{
      name: "Onboarding Email",
    content: "<p>hello </p>",
    }'
```

</RequestExample>
