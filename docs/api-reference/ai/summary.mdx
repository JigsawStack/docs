---
title: "Summary"
api: "POST https://api.jigsawstack.com/v1/ai/summary"
description: "Generate smarter text summaries with AI"
---

### Body

<ParamField body="text" type="string" required>
  The text to summarize. Maximum character is 300,000. Not required if
  `file_store_key` or `url` is specified.

The `text` parameter supports two types:

  <ul>
    <li>`string` - `{"text":"Your long text here"}`</li>
    <li>`Array<string>` - `{"text":["Long text 1", "Long text 2"]}`</li>
  </ul>
</ParamField>

<ParamField body="type" type="string" default="text">
  The summary result type. Supported values are:
  <ul>
    <li>`text` - Returns a text summary.</li>
    <li>`points` - Returns an array with points summary.</li>
  </ul>
</ParamField>

<ParamField body="url" type="string">
  The PDF url. Not required if `file_store_key` or `text` is specified.
</ParamField>

<ParamField body="file_store_key" type="string">
  The key used to store the PDF on Jigsawstack file
  [Storage](/docs/api-reference/store/file/add). Not required if `url` or `text`
  is specified.
</ParamField>

<ParamField body="max_points" type="number" default="2">
  The maximum number of points to return. Maximum is 100.
</ParamField>

<ParamField body="max_characters" type="number">
  The maximum number of characters to summarize text into
</ParamField>

<Snippet file="header.mdx" />

### Response

<ResponseField name="success" type="boolean">
  Indicates whether the call was successful.
</ResponseField>

<ResponseField name="summary" type="string | array<string>">
  The summarized text.
</ResponseField>

<RequestExample>

```javascript Javascript
import { JigsawStack } from "jigsawstack";

const jigsawstack = JigsawStack({
  apiKey: "your-api-key",
});

const result = await jigsawstack.summary({
  text: "The Leaning Tower of Pisa, or simply, the Tower of Pisa, is the campanile, or freestanding bell tower, of Pisa Cathedral.",
});
```

```python Python
from jigsawstack import JigsawStack

jigsawstack = JigsawStack(api_key="your-api-key")

result = jigsawstack.summary({"text": "The Leaning Tower of Pisa, or simply, the Tower of Pisa, is the campanile, or freestanding bell tower, of Pisa Cathedral." });
```

```bash cURL
curl --location --request POST 'https://api.jigsawstack.com/v1/ai/summary' \
--header 'Content-Type: application/json' \
--header 'x-api-key: <your-api-key>' \
--data '{
    "text": "Lorem Ipsum!"
}'
```

</RequestExample>
