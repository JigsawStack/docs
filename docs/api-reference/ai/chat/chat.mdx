### Body

<ParamField body="message" type="string" required>
  The chat message.
</ParamField>

<ParamField body="additional_context" type="string">
  Additional context for the message.
</ParamField>

<ParamField body="include_history" type="boolean" default="true">
  Use previous chat message history to determine a response. Defaults to `true`
</ParamField>

<ParamField body="include_context" type="boolean" default="true">
  Use chat session context (documents) to determine a response. Defaults to
  `true` A chat session context can be added by [attaching a
  document](/docs/api-reference/ai/chat/document/attach).
</ParamField>

<ParamField body="strict_context" type="boolean" default="false">
  Only rely on the context of the chat to determine a response.
</ParamField>
