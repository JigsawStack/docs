---
title: "Sentiment"
api: "POST https://api.jigsawstack.com/v1/ai/sentiment"
description: "Perform line by line sentiment analysis on any text."
---

### Body

<ParamField body="text" type="string" required>
  The text to be analyzed
</ParamField>

<Snippet file="header.mdx" />

### Response

<ResponseField name="success" type="boolean">
  Indicates whether the call was successful.
</ResponseField>

<ResponseField name="sentiment" type="object">
<Expandable title="Toggle object" defaultOpen="true">
<ResponseField name="sentiment" type="string">
  Indicates the sentiment the text. E.g `positive` or `negative`
</ResponseField>
<ResponseField name="emotion" type="string">
  Indicates the type of emotion inferred from the text. 
  Supported emotions:
  <ul>
  <li>`anger`</li>
  <li>`fear`</li>
  <li>`sadness`</li>
  <li>`happiness`</li>
  <li>`anxiety`</li>
  <li>`disgust`</li>
  <li>`embarrassment`</li>
  <li>`love`</li>
  <li>`surprise`</li>
  <li>`shame`</li>
  <li>`envy`</li>
  <li>`satisfaction`</li>
  <li>`self-confidence`</li>
  <li>`annoyance`</li>
  <li>`boredom`</li>
  <li>`hatred`</li>
  <li>`compassion`</li>
  <li>`guilt`</li>
  <li>`loneliness`</li>
  <li>`depression`</li>
  <li>`pride`</li>
  <li>`neutral`</li>
  </ul>
</ResponseField>
<ResponseField name="score" type="number">
  Indicates the sentiment score of the text.
</ResponseField>
<ResponseField name="sentences" type="Array<object>">
  The list of texts with associated sentiment result
</ResponseField>

</Expandable>

</ResponseField>

<RequestExample>

```javascript Javascript
import { JigsawStack } from "jigsawstack";

const jigsawstack = JigsawStack({
  apiKey: "your-api-key",
});

const result = await jigsawstack.sentiment({ text: "I am happy!" });
```

```python Python
from jigsawstack import JigsawStack

jigsawstack = JigsawStack(api_key="your-api-key")

 result = jigsawstack.sentiment({ text: "I am happy!" });
```

```bash cURL
curl --location --request POST 'https://api.jigsawstack.com/v1/ai/sentiment' \
--header 'Content-Type: application/json' \
--header 'x-api-key: <your-api-key>' \
--data '{
    "text": "I am happy!"
}'
```

</RequestExample>
