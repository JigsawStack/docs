### Query

<ParamField query="address" type="string" required>
  The crypto address to validate.
</ParamField>

<ParamField query="currency" type="string" required>
  The type of account to validate. Possible values are `ETH`, `BTC`.
  <ul>
    <li>ETH</li>
    <li>BTC </li>
  </ul>
</ParamField>

<Snippet file="header.mdx" />

### Response

<ResponseField name="success" type="boolean">
  Indicates whether the call was successful.
</ResponseField>

<ResponseField name="message" type="string">
  The message returned by the API.
</ResponseField>

<ResponseField name="valid" type="boolean">
  Indicates whether the address is valid.
</ResponseField>
