---
title: "Currency Conversion"
description: "Converts a currency to another currency using the [Exchange Rates API](/docs/api-reference/currency/convert)."
icon: "code"
---

## Initial Steps

- Setup an account (if you don't have an account already)
- Get your API key from [here](https://jigsawstack.com/dashboard).

## Usage

<CodeGroup>

```javascript Node.js
const convert = async () => {
  const endpoint = "http://api.jigsawstack.com/v1/currency/convert";

  const options = {
    headers: {
      "Content-Type": "application/json",
      "x-api-key": "<your-api-key-here>",
    },
  };

  const resp = await fetch(`${endpoint}?from=USD&to=NGN&amount=1000`, options);

  const data = await resp.json();
  console.log(data);
};
```

```bash cURL
curl -X GET "http://api.jigsawstack.com/v1/currency/convert?from=USD&to=NGN&amount=1000" \
     -H "Content-Type: application/json" \
     -H "x-api-key: <your-api-key-here>"
```

</CodeGroup>
