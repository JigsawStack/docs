---
title: "Website Screenshot"
description: "Capture website screenshots and receive output in both image and PDF formats using the [HTML To ANY API](/docs/api-reference/web/html-to-any)."
---

## Initial Steps

- Setup an account (if you don't have an account already)
- Get your API key from [here](https://jigsawstack.com/dashboard).

## Usage

<CodeGroup>

```javascript Node.js
import { JigsawStack } from "jigsawstack";

const jigsawstack = JigsawStack({
  apiKey: "your-api-key",
});

const params = {
  url: "https://jigsawstack.com",
  quality: 80,
  full_page: true,
  type: "jpeg",
  dark_mode: true,
};
const result = await jigsawstack.web.html_to_any(params);

const buffer = await result.arrayBuffer();

fs.writeFileSync(
  path.join(__dirname, `/screens/screenshot.jpeg`),
  Buffer.from(buffer)
);
```

```bash cURL
curl --location --request POST 'http://api.jigsawstack.com/v1/web/html_to_any'  --header 'Content-Type: application/json' --header 'x-api-key: <your-api-key-here>' \
--data '{
    url: "https://jigsawstack.com",
    quality: 80,
    full_page: true,
    type: "jpeg",
    dark_mode: true,
  }'
```
