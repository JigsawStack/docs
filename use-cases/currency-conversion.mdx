---
title: "Currency Conversion"
description: "Converts a currency to another currency using the [Exchange Rates API](https://docs.jigsawstack.com/api-reference/currency/convert)."
icon: "code"
---

## Initial Steps

- Setup an account (if you don't have an account already)
- Get your API key from [here](https://jigsawstack.com/dashboard).

## Usage

<CodeGroup>

```javascript Node.js
const apiUrl = "http://api.jigsawstack.com/v1";
const onConvert = async () => {
  try {
    const headers = {
      "Content-Type": "application/json",
      "x-api-key": "<your-api-key-here>",
    };

    const result = await fetch(
      `${apiUrl}/currency/convert?from=USD&to=NGN&amount=1000`,
      {
        headers,
      }
    );
    console.log(result);
  } catch (e) {
    handleError(e);
  }
};
```

```bash cURL
curl -X GET "http://api.jigsawstack.com/v1/currency/convert?from=USD&to=NGN&amount=1000" \
     -H "Content-Type: application/json" \
     -H "x-api-key: <your-api-key-here>"
```

</CodeGroup>
