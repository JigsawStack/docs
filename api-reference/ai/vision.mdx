---
title: "Vision"
api: "POST https://api.jigsawstack.com/v1/ai/vision"
description: "Recognise and describe objects within an image with great accuracy."
---

### Body

<ParamField body="url" type="string">
  The image url.
</ParamField>

<ParamField body="file_store_key" type="string">
  The key used to store the image on Jigsawstack File
  [Storage](https://docs.jigsawstack.com/api-reference/store/file/add).
</ParamField>

<ParamField body="ocr" type="boolean" defaultValue="true">
  When true, identifies all the objects within the image. Else, returns only
  describes the image.
</ParamField>

<ParamField body="prompt" type="string">
  The prompt used to describe the image.
</ParamField>

<Snippet file="header.mdx" />

### Response

<ResponseField name="success" type="boolean">
  Indicates whether the call was successful.
</ResponseField>
