---
title: "Spam Check"
api: "POST https://api.jigsawstack.com/v1/ai/spamcheck"
description: "This endpoint is used to perform spam check analysis"
---

### Body

<ParamField body="text" type="string" required>
  The text to check for spam.
</ParamField>

<Snippet file="header.mdx" />

### Response

<ResponseField name="success" type="boolean">
  Indicates whether the call was successful.
</ResponseField>

<ResponseField name="sentiment" type="string">
  The sentiment of the text.
</ResponseField>

<ResponseField name="check" type="object">

<Expandable title="Toggle object">

<ResponseField name="is_spam" type="boolean">
  Indicates whether the text is spam. If the text is not spam, this will be
  false. Default is true.
</ResponseField>

<ResponseField name="score" type="number">
  The spam score of the text. Default is 0. The lower the score, the less likely
  it will be spam
</ResponseField>

</Expandable>

</ResponseField>

<RequestExample>

```bash cURL
curl --location --request POST 'https://api.jigsawstack.com/v1/ai/spamcheck' \
--header 'Content-Type: application/json' \
--header 'x-api-key: <your-api-key>' \
--data '{
    "text": "I am happy!"
}'
```

```javascript Node.js
const endpoint = "https://api.jigsawstack.com/v1/ai/spamcheck";
const options = {
  method: "POST",
  headers: {
    "Content-Type": "application/json",
    "x-api-key": "<your-api-key>", // Replace with your actual API key.
  },
  body: JSON.stringify({
    text: "I am happy!",
  }),
};
const result = await fetch(endpoint, options);
const data = await result.json();
```
