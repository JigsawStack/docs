---
title: "Translate"
api: "POST https://api.jigsawstack.com/v1/ai/translate"
description: "Translate text from one language to another."
---

### Body

<ParamField body="current_language" type="string" required>
  The language code of the text to translate.
</ParamField>

<ParamField body="target_language" type="string" required>
  The language code of the text to translate to.
</ParamField>

<ParamField body="text" type="string" required>
  The text to translate.
</ParamField>

<Tip>Max text character is 5000.</Tip>

<Snippet file="header.mdx" />

### Response

<ResponseField name="success" type="boolean">
  Indicates whether the call was successful.
</ResponseField>

<ResponseField name="translated_text" type="text">
  The translated text.
</ResponseField>

<RequestExample>

```javascript Node.js
const endpoint = "https://api.jigsawstack.com/v1/ai/translate";
const options = {
  method: "POST",
  headers: {
    "Content-Type": "application/json",
    "x-api-key": "<your-api-key>", // Replace with your actual API key.
  },
  body: JSON.stringify({
    current_language: "en",
    target_language: "es",
    text: "Hello, world!",
  }),
};
const result = await fetch(endpoint, options);
const data = await result.json();
```
