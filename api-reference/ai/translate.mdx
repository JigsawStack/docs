---
title: "Translate"
api: "POST https://api.jigsawstack.com/v1/ai/translate"
description: "Translate text from one language to another."
---

### Body

<ParamField body="current_language" type="string">
  The language code of the text to translate.
</ParamField>

<ParamField body="target_language" type="string" required>
  The language code of the text to translate to.
</ParamField>

<ParamField body="text" type="string | string[]" required>
  The text to translate.
</ParamField>

<Tip>Max text character is 5000.</Tip>

<Snippet file="header.mdx" />

### Response

<ResponseField name="success" type="boolean">
  Indicates whether the call was successful.
</ResponseField>

<ResponseField name="translated_text" type="string | string[]">
  The translated text.
</ResponseField>

<RequestExample>

```javascript Javascript
import { JigsawStack } from "jigsawstack";

const jigsawstack = JigsawStack({
  apiKey: "your-api-key",
});

const result = await jigsawstack.translate({
  current_language: "en",
  target_language: "es",
  text: "Hello, world!",
});
```

```python Python
from jigsawstack import JigsawStack

jigsawstack = JigsawStack(api_key="your-api-key")

result = jigsawstack.translate({ text: "Hello, world!", target_language:"es", current_language:"en" });
```

```bash cURL
curl --location --request POST 'https://api.jigsawstack.com/v1/ai/translate' \
--header 'Content-Type: application/json' \
--header 'x-api-key: <your-api-key>' \
--data '{
    "current_language": "en",
    "target_language": "es",
    "text": "Hello, world!"
}'
```

</RequestExample>
