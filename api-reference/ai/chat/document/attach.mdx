---
title: "Attach Document"
api: "POST https://api.jigsawstack.com/v1/ai/chat/documents/attach"
description: "This endpoint is used to attach a document to a chat session."
---

### Body

<ParamField body="document_ids" type="array<string>">
  Document IDs to attach to a chat session
</ParamField>

<ParamField body="chat_session_id" type="string">
  Chat session ID to attach to a chat session
</ParamField>

<Snippet file="header.mdx" />

### Response

<ResponseField name="success" type="boolean">
  Indicates whether the call was successful.
</ResponseField>

<ResponseField name="attached_document_ids" type="string">
  Attached document IDs
</ResponseField>

<RequestExample>

```bash cURL
curl --location --request POST 'https://api.jigsawstack.com/v1/ai/chat/documents/attach' \
--header 'Content-Type: application/json' \
--header 'x-api-key: <your-api-key>' \
--data '{}'
```

```javascript Node.js
const endpoint = "https://api.jigsawstack.com/v1/ai/chat/document/attach";
const options = {
  method: "POST",
  headers: {
    "Content-Type": "application/json",
    "x-api-key": "<your-api-key>", // Replace with your actual API key.
  },
  body: JSON.stringify({
    document_ids: [],
    chat_session_id: "",
  }),
};
const result = await fetch(endpoint, options);
const data = result.json();
```
