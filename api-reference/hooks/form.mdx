---
title: "Form"
api: "POST https://api.jigsawstack.com/api/v1/hooks/form"
description: "This endpoint is used to generate form url to track form events."
---

### Body

<ParamField body="name" type="string" required>
  The name of the form.
</ParamField>

<ParamField body="webhook" type="string">
  The url to send the form data to.
</ParamField>

<ParamField body="redirect_url" type="string">
  The url to redirect to after the form is submitted.
</ParamField>

<ParamField body="meta" type="object">
  The meta data to send with the form data.
</ParamField>

<Snippet file="header.mdx" />

### Response

<ResponseField name="success" type="boolean">
  Indicates whether the call was successful.
</ResponseField>

```bash Example Request
curl --location --request POST 'https://api.jigsawstack.com/api/v1/hooks/form' \
--header 'Content-Type: application/json' \
--header 'x-api-key: <your-api-key>' \
--data '{
    "name": "My Form",
    "webhook": "https://example.com/webhook",
```
