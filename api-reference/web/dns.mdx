---
title: "DNS"
api: "GET https://api.jigsawstack.com/v1/web/dns"
description: "Get the DNS information of a domain."
---

### Query

<ParamField query="domain" type="string">
  The domain.
</ParamField>

<ParamField query="type" type="string">
  The DNS type. Supported values are:
  <ul>
    <li>`A`</li>
    <li>`NS`</li>
    <li>`MD`</li>
    <li>`MF`</li>
    <li>`CNAME`</li>
    <li>`SOA`</li>
    <li>`MB`</li>
    <li>`MG`</li>
    <li>`MR`</li>
    <li>`NULL`</li>
    <li>`WKS`</li>
    <li>`PTR`</li>
    <li>`HINFO`</li>
    <li>`MINFO`</li>
    <li>`MX`</li>
    <li>`TXT`</li>
    <li>`RP`</li>
    <li>`AFSDB`</li>
  </ul>

For the complete list visit, [DNS types](https://gist.github.com/iamNarcisse/0617fa14dcb39abf06fb00f524d20c14)

</ParamField>

<Snippet file="header.mdx" />

### Response

<ResponseField name="success" type="boolean">
  Indicates whether the call was successful.
</ResponseField>

```javascript Node.js
const endpoint =

const result = await fetch(endpoint);

```

```bash cURL
curl --location --request GET 'https://api.jigsawstack.com/v1/web/dns' \

```
