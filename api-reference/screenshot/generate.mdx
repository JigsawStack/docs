---
title: "Screenshot"
api: "POST https://api.jigsawstack.com/api/v1/screenshot"
description: "Takes a screenshot of a website."
---

### Body

<ParamField body="url" type="string" required>
  The URL of the website to take a screenshot of.
</ParamField>

<ParamField body="option" type="JSON">
  The options for the screenshot.
  <Expandable title="properties">
    <ResponseField name="quality" type="number">
      The quality of the screenshot. 0-100.
    </ResponseField>
    <ResponseField name="fullPage" type="boolean">
      Whether to take a full page screenshot.
    </ResponseField>
    <ResponseField name="omitBackground" type="boolean">
      Whether to omit the background.
    </ResponseField>
    <ResponseField name="type" type="string">
      The type of the screenshot. Can be either jpeg or png.
      <ul>
        <li>jpeg</li>
        <li>png</li>
      </ul>
    </ResponseField>
  </Expandable>
</ParamField>

### Response

<ResponseField name="success" type="boolean">
  Indicates whether the call was successful.
</ResponseField>

<RequestExample>

```bash Example Request
curl --location --request POST 'https://api.jigsawstack.com/api/v1/screenshot' \
--header 'Content-Type: application/json' \
--header 'x-api-key: <your-api-key>' \
--data '{
    "url": "https://jigsawstack.com"
}'
```

</RequestExample>

<ResponseExample>

```json Response
{
  "success": true
}
```

</ResponseExample>
