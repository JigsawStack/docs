---
title: "Timezone"
api: "GET https://api.jigsawstack.com/v1/geo/timezone"
description: "Get the timezone information for a given country, city or coordinate."
---

### Query

<ParamField query="country_code" type="string">
  The country code of the country to get the timezone information for. Not
  required if `city_code` is specified.
</ParamField>

<ParamField query="city_code" type="string">
  The city code of the city to get the timezone information for. Not required if
  `country_code` is specified.
</ParamField>

<ParamField query="lat" type="string">
  The the latitude to get the timezone. Required if `lng` is specified. Not
  required if `city_code` or `country_code` is specified.
</ParamField>

<ParamField query="lng" type="string">
  The longitude to get the timezone. Required if `lat` is specified. Not
  required if `city_code` or `country_code` is specified.
</ParamField>
