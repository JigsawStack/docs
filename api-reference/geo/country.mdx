---
title: "Country"
api: "GET https://api.jigsawstack.com/v1/geo/country"
description: "Get geo information for a country"
---

### Query

<ParamField query="country_code" type="string">
  The country code of the country. Filters result by country code.
</ParamField>

<ParamField query="currency_code" type="string">
  The currency code. Filters result by currency code.
</ParamField>

<ParamField query="city_code" type="string">
  The city code of the city. Filters result by city code.
</ParamField>

<ParamField query="search_value" type="string">
  The search value of the city.
</ParamField>

<ParamField query="lat" type="string">
  The latitude of the country. Required if `lng` is specified.
</ParamField>

<ParamField query="lng" type="string">
  The longitude of the country. Required if `lat` is specified.
</ParamField>

<ParamField query="limit" type="string" default="10">
  The maximum number of items to be returned.
</ParamField>

<Snippet file="header.mdx" />

### Response

<ResponseField name="success" type="boolean">
  Indicates whether the call was successful.
</ResponseField>

<RequestExample>

```javascript Node.js
import { JigsawStack } from "jigsawstack";

const jigsawstack = JigsawStack({
  apiKey: "your-api-key",
});

const result = await jigsawstack.geo.country({
  country_code: "SGP",
});
```

```python Python
from jigsawstack import JigsawStack

jigsawstack = JigsawStack(api_key="your-api-key")

result = jigsawstack.geo.country({"country_code":"SGP" });
```

```bash cURL
curl --location --request GET 'https://api.jigsawstack.com/v1/geo/country' \
--header 'x-api-key: <your-api-key>'
```

</RequestExample>
