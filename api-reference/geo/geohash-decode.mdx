---

### Query

<ParamField path="geohash" type="string" required>
  The geohash to decode
</ParamField>

<Snippet file="header.mdx" />

### Response

<ResponseField name="success" type="boolean">
  Indicates whether the call was successful.
</ResponseField>

<RequestExample>

```javascript Node.js
import { JigsawStack } from "jigsawstack";

const jigsawstack = JigsawStack({
  apiKey: "your-api-key",
});

const result = await jigsawstack.geo.geohash_decode("s14kv8brz");
```

```bash cURL
curl --location --request GET 'https://api.jigsawstack.com/v1/geo/geohash/{geohash}' \
--header 'x-api-key: <your-api-key>'
```
