---
title: "Distance"
api: "GET https://api.jigsawstack.com/v1/geo/distance"
description: "This endpoint is used to get the distance between two points on the earth."
---

### Query

<ParamField query="lat1" type="string" required>
  The latitude of the first point.
</ParamField>

<ParamField query="lat2" type="string" required>
  The latitude of the second point.
</ParamField>

<ParamField query="lng1" type="string" required>
  The longitude of the first point.
</ParamField>

<ParamField query="lng2" type="string" required>
  The longitude of the second point.
</ParamField>

<ParamField query="unit" type="string" default="K">
  The unit of measurement to use. Possible values are:
  <ul>
    <li>K - Kilometers</li>
    <li>N - Nautical Miles</li>
  </ul>
</ParamField>
