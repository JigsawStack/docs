---
title: "Geohash"
api: "GET https://api.jigsawstack.com/v1/geo/geohash"
description: "Get the geohash of a given latitude and longitude."
---

### Query

<ParamField query="lat" type="string" required>
  The latitude to encode
</ParamField>

<ParamField query="lng" type="string" required>
  The longitude to encode
</ParamField>

<ParamField query="precision" type="string" default="9" initialValue="9">
  The precision of the geohash. A higher precision value results in a shorter
  geohash string but represents a smaller area with more precision, while a
  lower precision value results in a longer geohash string but represents a
  larger area with less precision.
</ParamField>

<Snippet file="header.mdx" />

### Response

<ResponseField name="success" type="boolean">
  Indicates whether the call was successful.
</ResponseField>

<RequestExample>

```javascript Node.js
import { JigsawStack } from "jigsawstack";

const jigsawstack = JigsawStack({
  apiKey: "your-api-key",
});

const result = await jigsawstack.geo.geohash({
  lat: 9.838478,
  lng: 8.84847,
});
```

```bash cURL
curl --location --request GET 'https://api.jigsawstack.com/v1/geo/geohash' \
--header 'x-api-key: <your-api-key>'
```
