---
title: "Spell Check"
api: "GET https://api.jigsawstack.com/api/v1/validate/spell_check"
description: "Checks the spelling of a text."
---

### Query

<ParamField query="text" type="string" required>
  The text to check.
</ParamField>

<ParamField query="language_code" type="string" default="en" required>
  The language code of the text. Default is "en".
</ParamField>

### Header

<ParamField header="x-api-key" type="string" required>
  Your jigsawstack API key
</ParamField>

### Response

<ResponseField name="success" type="boolean">
  Indicates whether the call was successful.
</ResponseField>

<ResponseField name="message" type="string">
  The message returned by the API.
</ResponseField>

<ResponseField name="misspellings_found" type="number">
  The number of misspellings found in the text.
</ResponseField>

<ResponseField name="auto_correct_text" type="string">
  The auto corrected text.

If no misspellings were found, the original text is returned.

</ResponseField>

<RequestExample>

```bash Example Request
curl --location --request GET 'https://api.jigsawstack.com/api/v1/validate/spell_check' \
--header 'x-api-key: <your-api-key>'
```

</RequestExample>

<ResponseExample>

```json Response
{
  "success": true,
  "message": "Validated successfully",
  "misspellings_found": 0,
  "auto_correct_text": ""
}
```

</ResponseExample>
