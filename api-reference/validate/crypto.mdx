---
title: "Cryto Address"
api: "GET https://api.jigsawstack.com/v1/validate/crypto"
description: "This endpoint is used to validate a crypto address."
---

### Query

<ParamField query="address" type="string" required>
  The crypto address to validate.
</ParamField>

<ParamField query="currency" type="string" required>
  The type of account to validate. Possible values are `ETH`, `BTC`.
  <ul>
    <li>ETH</li>
    <li>BTC </li>
  </ul>
</ParamField>

### Header

<ParamField header="x-api-key" type="string" required>
  Your jigsawstack API key
</ParamField>

### Response

<ResponseField name="success" type="boolean">
  Indicates whether the call was successful.
</ResponseField>

<ResponseField name="message" type="string">
  The message returned by the API.
</ResponseField>

<ResponseField name="valid" type="boolean">
  Indicates whether the address is valid.
</ResponseField>

<RequestExample>

```bash Example Request
curl --location --request GET 'https://api.jigsawstack.com/v1/validate/crypto?address=""' \
--header 'x-api-key: <your-api-key>'
```

</RequestExample>

<ResponseExample>

```json Response
{
  "success": true,
  "message": "Validated successfully",
  "valid": true
}
```

</ResponseExample>
