---
title: "Credit Card"
api: "GET https://api.jigsawstack.com/api/v1/validate/creditcard"
description: "This endpoint is used to validate a credit card"
---

### Query

<ParamField query="credit_card_number" type="string" required>
  The credit card number to validate
</ParamField>

<ParamField query="get_image" type="boolean">
  Whether to return an image to display the credit card type
</ParamField>

### Header

<ParamField header="x-api-key" type="string" required>
  Your jigsawstack API key
</ParamField>

### Response

<ResponseField name="success" type="boolean">
  Indicates whether the call was successful.
</ResponseField>

<ResponseField name="message" type="string">
  The message returned by the API.
</ResponseField>

<ResponseField name="valid" type="boolean">
  Indicates whether the credit card number is valid.
</ResponseField>

<ResponseField name="image_data_uri" type="string">
  The image data URI of the credit card type.
</ResponseField>

<RequestExample>

```bash Example Request
curl --location --request GET 'https://api.jigsawstack.com/api/v1/validate/creditcard?credit_card_number=""&get_image=false' \
--header 'x-api-key: <your-api-key>'
```

</RequestExample>

<ResponseExample>

```json Response
{
  "success": true,
  "message": "Validated successfully",
  "valid": true,
  "image_data_uri": ""
  "provider": "none"
}
```

</ResponseExample>
