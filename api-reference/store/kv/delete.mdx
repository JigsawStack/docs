---
title: "Delete"
api: "DELETE https://api.jigsawstack.com/v1/store/kv/{key}"
description: "Delete a key-value pair from the store"
---

### Path

<ParamField path="key" type="string" required>
  The key of the key-value pair to delete.
</ParamField>

<Snippet file="header.mdx" />

### Response

<ResponseField name="success" type="boolean">
  Indicates whether the call was successful.
</ResponseField>

<ResponseField name="message" type="string">
  The message returned by the API.
</ResponseField>

<RequestExample>

```javascript Node.js
import { JigsawStack } from "jigsawstack";

const jigsawstack = JigsawStack({
  apiKey: "your-api-key",
});

const key = "jigsaw";

const result = await jigsawstack.store.kv.delete(key);
```

```bash cURL
curl --location --request DELETE 'https://api.jigsawstack.com/v1/store/kv/{key}' \
--header 'Content-Type: application/json' \
--header 'x-api-key: <your-api-key>'
```

</RequestExample>

<ResponseExample>

```json Response
{
  "success": true,
  "message": "Record deleted successfully"
}
```

</ResponseExample>
