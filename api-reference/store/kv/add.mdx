---
title: "Add"
api: "POST https://api.jigsawstack.com/v1/store/kv"
description: "Add data to KV store."
---

### Body

<ParamField body="key" type="string" required>
  The key to store the data under.
</ParamField>

<ParamField body="value" type="string" required>
  The value to store.
</ParamField>

<ParamField body="encrypt" type="boolean" default="false">
  Whether to encrypt the data at rest. If set to true, the data will be
  encrypted using the
  [AES-256](https://en.wikipedia.org/wiki/Advanced_Encryption_Standard)
  encryption algorithm. We recommend you set this value to true for sensitive
  data.
</ParamField>

<Snippet file="header.mdx" />

### Response

<ResponseField name="success" type="boolean">
  Indicates whether the call was successful.
</ResponseField>

<RequestExample>

```javascript Node.js
import { JigsawStack } from "jigsawstack";

const jigsawstack = JigsawStack({
  apiKey: "your-api-key",
});

const key = "italy";

const result = await jigsawstack.store.kv.add({
  key,
  value: "Leanning Tower of Pisa",
  encrypt: true,
});
```

```bash cURL
    curl --location --request POST 'https://api.jigsawstack.com/v1/store/kv' \
    --header 'Content-Type: application/json' \
    --header "x-api-key: <your-api-here>" \
    --data '{
    "key": "italy",
    "value": "Leaning Tower of Pisa"
    }'
```

</RequestExample>

```json Response
}
```

</ResponseExample>
