---
title: "Unit"
api: "GET https://api.jigsawstack.com/v1/convert/unit"
description: "This endpoint converts a unit to another unit."
---

### Supported Units

<Info>cm|ft|inch|km|m|meter|mi|mm</Info>

### Query

<ParamField query="from" type="string" required>
  The unit to convert from.

<ul>
  <li>cm</li>
  <li>ft</li>
  <li>inch</li>
  <li>km</li>
  <li>m</li>
  <li>meter</li>
  <li>mi</li>
  <li>mm</li>
</ul>

</ParamField>

<ParamField query="to" type="string" required>
  The unit to convert to.
  <ul>
    <li>cm</li>
    <li>ft</li>
    <li>inch</li>
    <li>km</li>
    <li>m</li>
    <li>meter</li>
    <li>mi</li>
    <li>mm</li>
  </ul>
</ParamField>

<ParamField query="value" type="string" required>
  The value to convert.
</ParamField>

<Snippet file="header.mdx" />

### Response

<ResponseField name="success" type="boolean">
  Indicates whether the call was successful.
</ResponseField>

<ResponseField name="unit" type="number">
  The converted unit.
</ResponseField>

<RequestExample>

```bash Example Request
curl --location --request GET 'https://api.jigsawstack.com/v1/convert/unit' \
--header 'x-api-key: <your-api-key>'
```

</RequestExample>

<ResponseExample>

```json Response
{
  "success": true,
```

</ResponseExample>
